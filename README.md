## Eye Color Detection
    
#### Usage 
    $ python eye-color.py --image_path=data/1.jpg

    Output: 

    **** Eyes Color Percentage ****
    Blue        :  0.0 %
    Blue Gray   :  0.0 %
    Brown       :  99.35 %
    Brown Gray  :  0.22 %
    Brown Black :  0.0 %
    Green       :  0.33 %
    Green Gray  :  0.11 %
